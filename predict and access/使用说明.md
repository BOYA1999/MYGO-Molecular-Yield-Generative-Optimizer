# 分子ADMET评估系统使用说明

## 概述

本系统使用机器学习/深度学习方法对生成的分子进行全面的ADMET（吸收、分布、代谢、排泄、毒性）评估，包括以下六个方面的预测：

1. **代谢预测** (Metabolism Prediction)
2. **血浆暴露** (Plasma Exposure)
3. **血脑屏障渗透性** (BBB Permeability)
4. **器官毒性预测** (Organ Toxicity)
5. **致畸致癌致突变预测** (TCM Prediction)
6. **半衰期预测** (Half-life Prediction)

## 功能特点

- 支持分段输出报告，避免内容过长
- 支持多种输出格式（Markdown、JSON、CSV、HTML）
- 支持单个分子和批量分子评估
- 支持机器学习模型和规则基础方法
- 提供预测结果汇总与建议

cd 预测评估

## 使用方法

### 方法1: 命令行工具

#### 评估单个分子

```bash
python evaluate_molecules.py -s "CCO" -o ./reports --name ethanol
```

#### 评估SMILES文件

```bash
python evaluate_molecules.py -f molecules.smi -o ./reports
```

#### 评估SDF文件

```bash
python evaluate_molecules.py -f molecules.sdf -o ./reports --model-dir ./models
```

#### 使用规则基础方法（不使用ML模型）

```bash
python evaluate_molecules.py -s "CCO" -o ./reports --no-ml
```

### 方法2: Python API

```python
from rdkit import Chem
from 预测评估.ensemble_predictor import EnsembleADMETPredictor
from 预测评估.segmented_report_generator import SegmentedReportGenerator

# 初始化预测器
ensemble_predictor = EnsembleADMETPredictor(
    model_dir=None,  # 可选：预训练模型目录
    use_ml=True      # True: 使用ML模型, False: 使用规则基础方法
)

# 初始化报告生成器
report_generator = SegmentedReportGenerator(ensemble_predictor)

# 评估分子（从SMILES）
smiles = "CCO"
mol = Chem.MolFromSmiles(smiles)

# 生成报告
report_paths = report_generator.generate_segmented_report(
    mol=mol,
    output_dir="./reports",
    mol_name="ethanol"
)

print(f"报告已生成: {report_paths}")
```

## 输出文件说明

评估完成后，会在输出目录生成以下文件：

### 分段Markdown报告
- `{mol_name}_section_01_header.md` - 报告头部和概述
- `{mol_name}_section_02_metabolism.md` - 代谢预测详细结果
- `{mol_name}_section_03_plasma_exposure.md` - 血浆暴露预测详细结果
- `{mol_name}_section_04_bbb.md` - 血脑屏障渗透性预测详细结果
- `{mol_name}_section_05_organ_toxicity.md` - 器官毒性预测详细结果
- `{mol_name}_section_06_tcm.md` - 致畸致癌致突变预测详细结果
- `{mol_name}_section_07_half_life.md` - 半衰期预测详细结果
- `{mol_name}_section_08_summary.md` - 综合总结和建议

### 完整报告
- `{mol_name}_full_report.md` - 完整的Markdown报告（包含所有部分）
- `{mol_name}_report.json` - JSON格式的详细报告
- `{mol_name}_report.csv` - CSV格式的表格报告
- `{mol_name}_report.html` - HTML格式的可视化报告

## 报告内容说明

### 1. 代谢预测 (Metabolism Prediction)

预测内容：
- 代谢稳定性（高/中/低）
- CYP450底物可能性
- 估计清除率
- 影响代谢稳定性的因素

### 2. 血浆暴露 (Plasma Exposure)

预测内容：
- 口服生物利用度
- 血浆蛋白结合率
- 最大血浆浓度（Cmax）
- 药时曲线下面积（AUC）

### 3. 血脑屏障渗透性 (BBB Permeability)

预测内容：
- BBB渗透性（是/否）
- 渗透性评分
- 脑/血浆比率

### 4. 器官毒性预测 (Organ Toxicity)

预测内容：
- 总体毒性评估
- 肝毒性风险
- 肾毒性风险
- 心脏毒性风险
- 神经毒性风险

### 5. 致畸致癌致突变预测 (TCM Prediction)

预测内容：
- 总体TCM风险
- 致畸性风险
- 致癌性风险
- 致突变性风险

### 6. 半衰期预测 (Half-life Prediction)

预测内容：
- 消除半衰期（小时）
- 半衰期分类（短/中/长）
- 清除率分类

## 综合评分说明

系统会计算一个综合ADMET评分（0.0-1.0），评分基于所有六个方面的预测结果：

- **≥0.7**: 优秀 - 分子具有良好的ADMET性质，适合进一步开发
- **0.4-0.7**: 中等 - 分子ADMET性质中等，可能需要优化
- **<0.4**: 较差 - 分子ADMET性质较差，需要显著优化

## 注意事项

1. **预测方法**: 系统支持机器学习模型和规则基础方法。如果未提供预训练模型，将自动使用规则基础方法。

2. **预测准确性**: 预测结果基于计算模型，仅供参考。实际药物开发中需要结合实验数据进行验证。

3. **模型文件**: 如需使用机器学习模型，请将模型文件放在指定目录中，格式为 `{predictor_name}_model.pkl`。

4. **文件格式支持**: 
   - SMILES文件: `.smi`, `.txt`（每行一个SMILES）
   - SDF文件: `.sdf`
   - MOL文件: `.mol`

## 工作流程

本系统遵循workflow3的工作流程：

1. **分析问题**: 理解评估需求，确定评估内容
2. **制定方案**: 选择合适的预测方法和参数
3. **执行方案**: 运行评估并生成报告

## 示例

### 示例1: 评估单个分子

```python
from rdkit import Chem
from 预测评估.ensemble_predictor import EnsembleADMETPredictor
from 预测评估.segmented_report_generator import SegmentedReportGenerator

# 初始化
ensemble = EnsembleADMETPredictor(use_ml=False)  # 使用规则基础方法
generator = SegmentedReportGenerator(ensemble)

# 评估
mol = Chem.MolFromSmiles("CCO")
paths = generator.generate_segmented_report(
    mol=mol,
    output_dir="./reports",
    mol_name="ethanol"
)
```

### 示例2: 批量评估

```python
from rdkit import Chem
from 预测评估.evaluate_molecules import evaluate_batch_molecules

# 准备分子列表
smiles_list = ["CCO", "CCN(CC)CC", "c1ccccc1"]
mols = [Chem.MolFromSmiles(s) for s in smiles_list]

# 批量评估
paths = evaluate_batch_molecules(
    mols=mols,
    output_dir="./reports",
    use_ml=False
)
```
